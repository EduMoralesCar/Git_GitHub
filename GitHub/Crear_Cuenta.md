
## Cómo crear una cuenta en GitHub

GitHub es una plataforma para alojar proyectos y colaborar usando Git. Para empezar, necesitas crear una cuenta gratuita.

### Pasos para crear una cuenta en GitHub

1. Ingresa a la página oficial: [https://github.com/join](https://github.com/join)
2. Completa el formulario:
	- Escribe tu correo electrónico.
	- Elige un nombre de usuario único.
	- Crea una contraseña segura.
3. Haz clic en "Create account".
4. Verifica tu correo electrónico siguiendo el enlace que te envía GitHub.
5. Completa el proceso de registro (puedes omitir preguntas opcionales).

### Recomendaciones de seguridad
- Usa una contraseña fuerte y única.
- Activa la verificación en dos pasos (2FA) desde la configuración de tu cuenta para mayor seguridad.

¡Listo! Ahora puedes crear repositorios, colaborar y explorar proyectos en GitHub.
