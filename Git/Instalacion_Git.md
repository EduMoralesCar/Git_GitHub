
## Instalación de Git

### 1. Descargar Git

Puedes descargar la última versión de Git desde el sitio oficial:

[Descargar Git](https://git-scm.com/downloads)

### 2. Instalar en Windows

1. Ejecuta el instalador descargado.
2. Sigue los pasos del asistente, dejando las opciones por defecto si no tienes preferencias específicas.
3. Al finalizar, Git estará disponible en tu sistema.

### 3. Verificar la instalación

Abre una terminal (PowerShell, CMD o Git Bash) y ejecuta:

```bash
git --version
```

Si ves la versión instalada, Git está listo para usarse.

### 4. Configuración inicial recomendada

Configura tu nombre y correo para los commits:

```bash
git config --global user.name "Tu Nombre"
git config --global user.email "tu@email.com"
```
